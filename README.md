# sito-mui-error-component

## Component to catch error with react-error-boundary

### Warning!!! It requires @mui libs on main project

How to use it?

```
import ErrorBoundary from "sito-mui-error-component;

const App = () => {
    return <ErrorBoundary>
        <h1>Hola mundo!</h1>
    </ErrorBoundary>
}

```
